# vue-music

麻烦路过的小可爱动动你小手点个star

本项目仅供个人学习使用！

## 项目介绍

#### 技术栈

vue全家桶

element-ui组件库

axios

[simple-scroll](https://github.com/liaoruikang/vue-simple-scroll) （自己封装的滚动插件）

simple-video （自己封装的视频播放插件，暂无仓库）

#### 项目功能

- 登录(扫码，手机号，验证码，邮箱) 
- 退出登录
- 首页推荐 轮播图
- 歌单排行榜，全部歌单 歌单详情
- 电台 电台详情
- 歌手列表详情
- 用户详情
- 专辑列表详情
- 播放音乐 歌词滚动 播放模式切换
- 收藏音乐，歌单，专辑，mv，节目，电台
- 分享音乐，歌单，专辑，mv，节目，电台
- @好友
- 评论（不包括发表评论）
- 播放mv 视频
- 我的音乐 修改歌单 
- 歌单封面上传
- 搜索

#### UI界面

本项目ui完全复刻网易云pc端ui

## 启动项目

```
npm install # 安装依赖
npm install # 启动项目
```

浏览器访问[localhost:8080](http://localhost:8080)就启动成功了
